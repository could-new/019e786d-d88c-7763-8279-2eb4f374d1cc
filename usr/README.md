# Generator Paper Ilmiah

Aplikasi Flutter ini berfungsi untuk men-generate (membuat) dan melihat paper ilmiah (PDF) dengan judul "Analisis Rantai Nilai (Value Chain) Kopi dan Evaluasi Kritis Fair Trade dalam Distribusi Margin Petani".

## Fitur
- Pembuatan dokumen PDF langsung di perangkat (on-device) tanpa server.
- Review paper ilmiah secara real-time dengan tampilan preview PDF.
- Kemampuan untuk mencetak, membagikan, dan mengunduh hasil PDF.

## Cara Menjalankan
1. Pastikan Flutter SDK telah terinstal (versi 3.0 ke atas).
2. Jalankan `flutter pub get` untuk mengunduh dependencies.
3. Jalankan `flutter run -d chrome` untuk melihat versi Web, atau pilih device target lainnya.

## Tentang CouldAI
Aplikasi ini di-generate menggunakan [CouldAI](https://could.ai), sebuah AI app builder untuk aplikasi cross-platform yang mengubah prompt menjadi aplikasi native untuk iOS, Android, Web, dan Desktop. CouldAI menggunakan agen AI otonom yang merancang, membangun, menguji, mendeploy, dan mengiterasi aplikasi yang siap untuk produksi.
